# Bridge Learning Tech - Practical Assignment
 
![](https://github.com/Rhys096/Bridge-Learning-Tech---Practical-Assignment/blob/main/game.png)

As the player collects more pickups, the levels advance and the players size decreases. Alternate between spheres and capsules to increase a combo multiplier that will maximise your score. Be careful; cubes fall more rapidly over time.

All content built from scratch. Carpet texture taken from www.sharetextures.com
